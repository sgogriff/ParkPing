# Install On Android Phone

This guide is for people who want to install ParkPing from the project's GitHub Releases page.

## Before you start

**You do not need to sign the app yourself**. The APK should already be signed before it is uploaded to GitHub. You only need to download it and install it.

## Install steps

1. Open the ParkPing GitHub page on your phone.
2. Tap `Releases`.
3. Open the latest release.
4. Download the APK file. It will usually be called something like `ParkPing.apk`.
5. When the download finishes, tap the file to open it.
6. If Android says installation is blocked, allow installs from your browser or Files app when prompted.
7. Open the APK again.
8. Tap `Install`.
9. When installation finishes, tap `Open`.

## First launch

When you open ParkPing for the first time, the app will guide you through setup.

It may ask for:

- notifications
- location
- background location

These permissions are needed so the app can show reminders when you arrive at work.

## Updating later

To install a newer version:

1. Go back to the GitHub `Releases` page.
2. Download the newest APK.
3. Open it and install it.

Your phone should update the app and keep your existing settings if available.

## If Android shows a warning

Because ParkPing is installed from GitHub and not from the Play Store, Android may show a warning before installation. That is normal for manually installed apps.

Only continue if:

- you downloaded the APK from the official ParkPing GitHub repository
- you trust the person who published it (that's me!)... or have enough free time to read the source code.

## If installation is blocked

Try these steps:

1. Make sure you downloaded the APK from `Releases`, not the source code zip... Make sure the file name ends in `.apk`.
2. Open the file again from your browser downloads list or Files app.
3. If prompted, allow that app to install unknown apps.
4. Try the install again.

## Uninstalling ParkPing

You can remove the app at any time:

1. Open `Settings`
2. Tap `Apps`
3. Tap `ParkPing`
4. Tap `Uninstall`

This removes the app and it's data from your phone.
